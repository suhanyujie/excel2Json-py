# excel2Json-py
提供给运营或者游戏策划的工具，将配置文件 —— xlsx 文件转换为 json 文件，从而把 json 配置文件提供给开发使用

## 程序截图

![](./docs/images/gui-show1.png)

## 增加配置文件
* 在程序目录下新建 `settings.ini` 配置文件，参考 [`settings.example.ini`](./settings.example.ini)
